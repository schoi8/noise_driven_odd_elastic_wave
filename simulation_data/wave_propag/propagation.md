Data used for observation of wave propagation behavior with space-dependent noise. They were analyzed by the code toymodel_wave_propag_analysis.ipynb.

1run data are for generation of a movie showing the propagating behavior. They were generated by the code toymodel_wave_propag.c.

100runs data are for generation of the dispersion relation plots. They were generated by the code toymodel_wave_propag_multirun.c.
