Current correlation data used to determine the phase boundary for the wave regime with self-circling frequency w=1, transverse force magnitude f0=1 and vsig=0.02. 
Other parameters are provided in simulation_para.txt. 
The data were generated by the code starfishmodel_phasediag_wave_multirun.c and were analyzed by the code starfishmodel_phasediag_wave_analysis.ipynb.

Average over 100 runs was used to determine the phase boundaries while average over 1000 runs was used to generate the plot for the dispersion relations.
